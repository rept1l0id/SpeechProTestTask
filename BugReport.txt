1. Регистрация при отсутствии пароля
    Окружение: контейнер vm_docker_ui (https://github.com/doublebon/Ui_Test_Task/blob/main/vm_docker_ui.7z)
    Предусловие: контейнер запущен, в базу данных пользователей не делалось записей с пустыми полями пароля
    Шаги воспроизведения:
        1. Открываем страницу http://localhost:5000
        2. Переходим на страницу регистрации (http://localhost:5000/signup)
        3. Вводим в форму регистрации только почту
        4. Нажимаем кнопку "Sign Up"
    Ожидание: получение сообщения об ошибке в форме
    Фактический результат: запись пользователя добавлена в БД без пароля, имеется возможность пройти
    авторизацию для данного пользователя

2. Регистрация при отсутствии e-mail
    Окружение: контейнер vm_docker_ui (https://github.com/doublebon/Ui_Test_Task/blob/main/vm_docker_ui.7z)
        Предусловие: контейнер запущен, в базу данных пользователей не делалось записей с пустыми полями e-mail
        Шаги воспроизведения:
            1. Открываем страницу http://localhost:5000
            2. Переходим на страницу регистрации (http://localhost:5000/signup)
            3. Вводим в форму регистрации только пароль
            4. Нажимаем кнопку "Sign Up"
        Ожидание: получение сообщения об ошибке в форме
        Фактический результат: запись пользователя добавлена в БД без почты, имеется возможность пройти
